### Step 2: Go to Settings Tab

Click on the "Settings" tab in the admin dashboard.

### Step 3: Update Restaurant Branding

You'll see a section called "Restaurant Branding" with these fields:

#### 1. Restaurant Name
- **What it does:** Updates the browser tab title and navigation header
- **Example:** "Simoo Restaurant", "Café Milano", "Burger House"
- **Where it appears:**
  - Browser tab title
  - Navigation bar (large text)
  - Throughout the menu

#### 2. Restaurant Subname
- **What it does:** Subtitle/tagline under the restaurant name
- **Example:** "Authentic Ethiopian Cuisine", "Fresh & Delicious"
- **Where it appears:**
  - Under the restaurant name in navigation

#### 3. Restaurant Logo
- **What it does:** Updates the favicon and navigation logo
- **Format:** PNG, JPG, or SVG
- **Recommended size:** 512x512px (square)
- **Where it appears:**
  - Browser tab icon (favicon)
  - Navigation bar (circular icon)
  - Menu header

#### 4. Primary Color
- **What it does:** Changes the theme color throughout the site
- **Format:** Hex color code (e.g., #d97706)
- **Where it appears:**
  - Buttons
  - Category badges
  - Highlights
  - Gradients
  - Active states

### Step 4: Save Changes

Click "Update Settings" button at the bottom.

### Step 5: Refresh Your Browser

1. Go to https://simoo.skoolific.com
2. Press `Ctrl + F5` (hard refresh) to clear cache
3. You should see:
   - ✅ New restaurant name in browser tab
   - ✅ New logo as favicon
   - ✅ New logo in navigation
   - ✅ New colors throughout

## 🖼️ Logo Requirements

### Best Practices:
- **Format:** PNG with transparent background (recommended)
- **Size:** 512x512px or larger (square)
- **File size:** Under 500KB
- **Style:** Simple, recognizable icon
- **Colors:** Works well on both light and dark backgrounds

### What Makes a Good Logo:
- ✅ Simple and clean design
- ✅ Recognizable at small sizes
- ✅ Works in circular frame
- ✅ High contrast
- ✅ Represents your brand

### What to Avoid:
- ❌ Too much detail
- ❌ Text-heavy logos
- ❌ Very thin lines
- ❌ Low resolution images
- ❌ Complex gradients

## 🐛 Troubleshooting

### Logo Not Showing?
1. Make sure image is uploaded successfully
2. Check file format (PNG, JPG, SVG)
3. Verify file size is under 5MB
4. Hard refresh browser (Ctrl + F5)

### Favicon Not Updating?
1. Clear browser cache completely
2. Close and reopen browser
3. Try in incognito/private mode
4. Wait a few minutes for cache to clear

### Colors Not Changing?
1. Make sure you saved settings
2. Hard refresh the page
3. Check if hex color code is valid
4. Try a different color to test

### Title Not Updating?
1. Verify restaurant name is saved
2. Refresh the page
3. Check browser console for errors
